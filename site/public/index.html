Two jobs, one binary

Spend fewer tokens. Then get your agents out of each other's way.

agentflare is an optimization layer bolted to a coordination layer. The first makes one agent cheaper to run. The second lets several of them — or several sessions over time — work the same repo without colliding.

Layer 01

Optimize

Cut the token and cost overhead of a single agent session — inside the session and across sessions.

lean-ctxCompresses tool I/O — reads, shell output, search — up to 99%, in-session.
memoryDecisions, facts, and preferences that survive a session ending. SQLite + FTS5, ships in the binary.
CavemanTrims conversation verbosity ~65% (Claude Code).
PonytailStops agents over-engineering the code they write (Claude Code).
Layer 02

Coordinate

A local-first, no-daemon backend for running more than one agent — exposed as MCP tools any MCP-capable agent can call.

work itemsA shared backlog. Claim one before working it so two agents don't collide.
reviewFindings against a diff, consensus across reviewers, scores over time.
artifactsVersioned specs, plans, and docs — a durable handoff surface, not scratch files.
handoffPass addressed, threaded context when work moves between agents.
coachingSmall persistent nudges surfaced to an agent, only when relevant.
↔

They're different axes, not substitutes. lean-ctx saves tokens inside a session; memory saves the re-explaining tax across sessions; coordination keeps multiple agents out of each other's way. Everything is local-first, SQLite-backed, no daemon — over the same stdio MCP transport.

One command, run once

Install the binary. Wire your agent. Done.

A single static binary — the only runtime dependency is agentflare itself. Detection-first: already-satisfied pieces are skipped, nothing gets clobbered.

Linux / macOS — prebuilt, checksum-verified 
curl -fsSL https://raw.githubusercontent.com/getappz/agentflare/master/install.sh | sh
Homebrew — macOS / Linux 
brew tap getappz/agentflare
brew install agentflare
Scoop — Windows 
scoop bucket add agentflare https://github.com/getappz/agentflare
scoop install agentflare
Cargo — any platform with Rust 
cargo install --git https://github.com/getappz/agentflare
Then set up your agent — the run is the consent 
# writes hooks straight into the host's own settings — no marketplace
agentflare init --agent claude-code   # or: cursor · windsurf · vscode-copilot · cline · continue
✓ rules written · lean-ctx installed · hooks + MCP wired · memory ready
